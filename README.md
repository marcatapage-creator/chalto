# Chalto

Chalto est un copilote financier conçu pour les indépendants, qui simplifie la gestion de leur trésorerie, de leurs impôts et de leurs charges sociales.

L’outil permet de comprendre en temps réel combien vous pouvez dépenser, combien vous devez mettre de côté, et à quelles échéances vous devrez payer (URSSAF, impôts, retraite), même avec des revenus irréguliers.

Son objectif : transformer la complexité fiscale en décisions simples, visuelles et actionnables, pour réduire le stress financier et piloter son activité avec sérénité.

---

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
